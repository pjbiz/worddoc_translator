# 📄 Word 文档翻译工具

## ⚡ 快速开始（TLDR）

**Windows 用户：**
1. 下载本项目
2. 双击 `run.bat` 文件
3. 在浏览器中输入您的 OpenAI API 密钥
4. 开始翻译！

就这么简单！无需任何编程知识。

---

一个简单易用的工具，使用 ChatGPT 将英文 Word 文档（.docx）翻译成简体中文。

---

## ✨ 功能特点

- ✅ 翻译整个 Word 文档（包括段落和表格）
- ✅ **保留原始格式**（字体、样式、表格布局）
- ✅ 支持**批量上传多个文件**
- ✅ 自动将英文文件名翻译成中文
- ✅ 简洁的网页界面，无需编程知识
- ✅ 使用 OpenAI GPT-4o-mini 模型，性价比高

## 📋 使用前准备

在开始之前，您需要准备两样东西：

### 1️⃣ 安装 Python

Python 是运行此工具所需的程序。

**检查是否已安装：**
1. 按下键盘上的 `Win + R` 键
2. 输入 `cmd` 然后按回车
3. 在黑色窗口中输入 `python --version` 然后按回车
4. 如果显示类似 `Python 3.x.x` 的文字，说明已安装

**如果未安装，请按以下步骤安装：**

1. 打开浏览器，访问：https://www.python.org/downloads/
2. 点击黄色的 **"Download Python"** 按钮
3. 运行下载的安装文件
4. ⚠️ **重要**：在安装界面，务必勾选 **"Add Python to PATH"**（添加到系统路径）
5. 点击 **"Install Now"** 完成安装

---

### 2️⃣ 获取 OpenAI API 密钥

API 密钥是使用 ChatGPT 翻译服务的"通行证"。以下是详细获取步骤：

#### 第一步：注册/登录 OpenAI 账户

1. 打开浏览器，访问：https://platform.openai.com
2. 如果没有账户，点击 **"Sign up"**（注册）
   - 可以使用邮箱注册，或使用 Google/Microsoft 账户登录
   - 按照提示完成手机号验证
3. 如果已有账户，点击 **"Log in"**（登录）

#### 第二步：添加付款方式（首次使用需要）

1. 登录后，点击右上角的头像
2. 选择 **"Billing"**（账单）
3. 点击 **"Add payment method"**（添加付款方式）
4. 输入信用卡信息（支持 Visa、Mastercard 等）
5. 建议设置使用限额，避免意外超支：
   - 在 Billing 页面找到 **"Usage limits"**（使用限额）
   - 设置每月最大花费，例如 $10

#### 第三步：创建 API 密钥

1. 点击左侧菜单的 **"API keys"**，或直接访问：https://platform.openai.com/api-keys
2. 点击 **"Create new secret key"**（创建新密钥）
3. 给密钥起个名字（可选），例如："翻译工具"
4. 点击 **"Create secret key"**
5. ⚠️ **重要**：立即复制显示的密钥！
   - 密钥只显示一次，关闭后无法再次查看
   - 密钥格式类似：`sk-proj-xxxxxxxxxxxxxxxx`
   - 将密钥保存在安全的地方

#### 💰 费用说明

- OpenAI 按使用量收费
- GPT-4o-mini 模型非常便宜：约 $0.15 / 100万字符
- 翻译一份普通文档通常只需几分钱
- 建议设置月度限额以控制支出

## 🚀 快速开始

### Windows 用户（推荐方法）

1. **下载本工具**
   - 点击本页面的绿色 **"Code"** 按钮
   - 选择 **"Download ZIP"**
   - 解压下载的压缩包到任意文件夹

2. **运行工具**
   - 打开解压后的文件夹
   - 双击 **`run.bat`** 文件
   - 首次运行会自动安装所需组件（可能需要几分钟）
   - 安装完成后，浏览器会自动打开

3. **开始翻译**
   - 在网页中输入您的 OpenAI API 密钥
   - 点击验证按钮
   - 上传要翻译的 Word 文档
   - 点击"翻译"按钮
   - 下载翻译完成的文档

### Mac / Linux 用户

1. 下载并解压本工具

2. 打开终端（Terminal），进入工具文件夹：
   ```bash
   cd 下载路径/File-translator
   ```

3. 运行启动脚本：
   ```bash
   chmod +x run.sh
   ./run.sh
   ```

4. 浏览器会自动打开，按照网页提示操作

## 📖 使用说明

### 步骤一：输入 API 密钥

1. 打开工具后，首先看到 API 密钥输入界面
2. 将您之前复制的 API 密钥粘贴到输入框
3. 点击 **"验证并保存密钥"** 按钮
4. 等待验证成功（显示绿色勾号）

### 步骤二：上传文档

1. 点击 **"上传文档"** 区域
2. 选择一个或多个 .docx 文件
3. 支持同时上传最多 20 个文件
4. 每个文件最大 10 MB

### 步骤三：翻译文档

1. 确认上传的文件列表正确
2. 点击 **"翻译所有文档"** 按钮
3. 等待翻译完成（进度条会显示进度）
4. 大文档可能需要几分钟

### 步骤四：下载结果

1. 翻译完成后，会显示下载选项
2. **单个文件**：点击对应的下载按钮
3. **多个文件**：点击 "下载全部（ZIP）" 获取压缩包
4. 文件名也会自动翻译成中文

## ❓ 常见问题

### 问：提示"API 密钥无效"怎么办？

**答：** 请检查以下几点：
1. 密钥是否完整复制（以 `sk-` 开头）
2. 密钥前后是否有多余的空格
3. OpenAI 账户是否已添加付款方式
4. 账户余额是否充足

### 问：翻译速度很慢怎么办？

**答：** 这取决于：
1. 文档大小（大文档需要更多时间）
2. 网络速度
3. OpenAI 服务器负载

建议耐心等待，进度条会显示当前进度。

### 问：翻译后格式丢失怎么办？

**答：** 本工具会尽量保留原始格式。如果遇到问题：
1. 确保原文件是 .docx 格式（不是 .doc）
2. 复杂格式可能无法100%保留
3. 可以在翻译后手动调整

### 问：如何更换 API 密钥？

**答：** 
1. 在网页上点击 **"更换 API 密钥"** 按钮
2. 输入新的密钥即可

### 问：费用怎么计算？

**答：** 
- 按翻译的字符数收费
- GPT-4o-mini 模型约 $0.15 / 100万输入字符
- 翻译一份10页文档约 $0.01-0.05
- 在 OpenAI 平台可以查看详细使用记录
